1) File --> new --> project.
2) Project Name: fel case beta3etna de kol el project names el hane3melha hateb2a esm el assignment, zy assignment_1 masalan.s
3) Create New File.
4) File Name: esm el module el enta shaghal 3leeh.
5) Add file as type: Verilog.

Watch the attached video, it is very helpful. Notice that we will not probably use "force" as he has done in the video because we will usually test our
code using testbenches, so the testbench code written will assign the values we want to the different signals, we do not need to force them to get a certain
value.

Watch video 2.

It is also useful to change the color of some signals on the waveform to be distinguishable. To do so, right click on the signal you want to change its
color, then click properties, and change the color.

To save a set of waveform that you have created:
File --> save format --> wave.do
then write in the terminal below: do wave.do
