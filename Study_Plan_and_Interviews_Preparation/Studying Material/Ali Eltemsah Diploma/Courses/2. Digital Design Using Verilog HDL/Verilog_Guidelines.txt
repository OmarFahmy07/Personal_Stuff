Some checks you must do for any Verilog code you write:

1) Make sure that you are using non-blocking assignments inside all sequential always blocks, blocking assignments inside all combinational always blocks 
and assign statements.

2) Make sure that you are putting an initial value for all combinational always blocks including if or case statements. For if statements, make sure you are
using else at the end. For case statements, make sure you are using default at the end.

3) Make sure that all blocks' outputs are registered to avoid combinational loops (you may not need to register control signals like enable and load).
